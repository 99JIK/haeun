C++로 코테를 풀 때 STL을 모르면 시간 안에 못 푼다. 자주 쓰는 컨테이너와 알고리즘 함수를 정리했다.

<!-- truncate -->

## 자주 쓰는 컨테이너

### vector

동적 배열. 거의 모든 문제에서 사용.

```cpp
vector<int> v = {3, 1, 4, 1, 5};
v.push_back(9);
v.pop_back();
sort(v.begin(), v.end());
```

### map / unordered_map

key-value 저장소. `map`은 정렬 O(log n), `unordered_map`은 해시 O(1).

```cpp
unordered_map<string, int> freq;
for (auto& word : words) {
    freq[word]++;
}
```

### priority_queue

최대 힙이 기본. 최소 힙은 `greater`를 넣어줘야 함.

```cpp
// 최소 힙
priority_queue<int, vector<int>, greater<int>> pq;
pq.push(3);
pq.push(1);
pq.top(); // 1
```

### set

중복 없는 정렬된 집합. 원소 존재 여부 확인이 O(log n).

```cpp
set<int> s = {3, 1, 4};
s.insert(1);  // 중복이라 무시됨
s.count(3);   // 1 (있음)
```

## 자주 쓰는 알고리즘 함수

```cpp
#include <algorithm>

// 정렬
sort(v.begin(), v.end());
sort(v.begin(), v.end(), greater<int>()); // 내림차순

// 이분 탐색 (정렬된 상태에서)
binary_search(v.begin(), v.end(), target);
lower_bound(v.begin(), v.end(), target);

// 최대/최소
*max_element(v.begin(), v.end());
*min_element(v.begin(), v.end());

// 중복 제거
sort(v.begin(), v.end());
v.erase(unique(v.begin(), v.end()), v.end());

// 순열
sort(v.begin(), v.end());
do {
    // v의 모든 순열에 대해 실행
} while (next_permutation(v.begin(), v.end()));
```
